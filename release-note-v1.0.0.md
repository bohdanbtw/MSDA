# MSDA 1.0.0

*Mobile Steam Desktop Authenticator with a Visual Studio-first Android workflow.*

### 🔐 Authentication and Accounts
- Multi-account `.mafile` import and account hub
- Account details page with active 2FA code and countdown
- Tap-to-copy 2FA code to clipboard with user feedback
- Steam LoginV2 re-login flow for session refresh

### ✅ Confirmations
- Load active trade and market confirmations
- Bundle-level accept/decline functionality
- Per-item accept/decline actions
- Item icons displayed in confirmation rows
- Improved list stability and expansion behavior

### 🔔 Background and Notifications
- Background confirmation checks (settings controlled)
- Push notifications for pending confirmations (settings controlled)
- Notification permission handling for modern Android
- Scheduling successfully restored after device reboot or app update

### 🎨 UI and Theming
- Seamless hub and account detail navigation flow
- Dedicated settings screen
- Theme modes: System / Light / Dark
- Light and dark mode readability improvements
- Updated official app name to **MSDA**

### 🚀 Branding and Release
- Launcher icon and in-app logo assets successfully integrated
- Versioned APK output naming applied: `MSDA-1.0.0.apk`
- Legacy Nebula-named files and internal references removed
- Documentation updated (including `README.md`, setup guide, and architecture)
